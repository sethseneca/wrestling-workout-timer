# Audio Assets

- `whistle-start.m4a` - trimmed AAC conversion built from Freesound sound `218318` by SpliceSound, licensed Creative Commons 0.
- `three.m4a`, `two.m4a`, `one.m4a` - short male voice cue files generated locally for this app.
- `rest-horn.m4a`, `final-horn.m4a` - trimmed AAC conversions built from BigSoundBank Horn #6 (`2525`), licensed CC0/public domain.
- `ten-second-pop.m4a` - trimmed AAC conversion built from BigSoundBank Clapperboard (`1011`), licensed CC0/public domain.
